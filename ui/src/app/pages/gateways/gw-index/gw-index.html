Gateway lets you expose your local / edge server (those without public IP or port-forwarding) to the Internet.

How Does Gateway Work?

A Gateway is an Internet-facing WireGuard peer that serves as a proxy for your local and edge servers. Use any WireGuard supported VPN network.

Key Features

  • Advanced Global Reach: Utilize GCP's global load balancer with a single, anycast IP.
  • Comprehensive Protocol Support: Full compatibility with Layer 7 and Layer 4 (TCP/UDP) on any port.
  • Easy Access: Connect through subdomains or use custom domains and private IP addresses.
  • Secure: Automatic TLS certificates
  • Network protection with Cloudflare.
  • Open-Source and Privacy-Focused: Your node. We merely manage it.
